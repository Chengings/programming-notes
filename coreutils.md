`arch`: print machine architecture.

`df`: disk free. Shows the usage and free space on the mounted filesystems.

`nl`: number lines
```shell
$ nl foo
     1  Wolves enjoy waterballoons.
     2  Don't eat the wrong fruit.
     3  My skull is made of cheese.
     4  Welcome to the pain castle.
```

`pr`: paginate or columnate files for printing.

`tac`: print files in reverse. cat -> tac

`timeout`: run a command with time limit.
```shell
timeout 1m ping google.com      # run ping for a minute.
```
